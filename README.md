# brewfiles

## Install

- Xcodeをインストール
- 1PasswordをAppStoreからインストール(iCloud同期)
- install.shを実行

``` bash
$ ./install.sh
$ ./karabiner.sh
$ ln -s /usr/local/share/git-core/contrib/diff-highlight/diff-highlight /usr/local/bin
```
